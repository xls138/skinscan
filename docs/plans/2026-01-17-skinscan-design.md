# SkinScan - 颜值气质测试互动机

## 产品定位

**场景**: 户外拍照互动机，商家引流  
**目标**: 娱乐性质，产生即时社交价值  
**核心**: 让用户想拍照分享结果

---

## 商业模式

| 层级 | 内容 | 解锁方式 |
|-----|------|---------|
| **免费卡片** | 气质标签 + 年龄 + 颜值分 + 一句话评语 | 扫脸即得 |
| **付费详情** | 雷达图 + 6 维度分析 + 护肤建议 | 付费解锁 |

## 分享卡片 (免费层)

### 卡片内容
- 用户照片
- 气质标签 (如: "清冷系 · 白月光")
- 预测年龄 (如: "AI 测龄: 22 岁")
- 颜值评分 (如: "颜值 92 分")
- 一句话评语 (根据气质类型匹配风格)

### 评语风格对应

| 气质类型 | 评语风格 | 示例 |
|---------|---------|------|
| 清冷系 | 诗意型 | "眉间疏朗，自有一种不染尘埃的清贵气" |
| 甜美系 | 夸赞型 | "笑起来像草莓味的夏天，让人想要保护" |
| 御姐系 | 网感型 | "这张脸写满了'别惹我但可以爱我'，绝" |
| 氛围感系 | 诗意型 | "眼里藏着故事，嘴角挂着秘密，氛围感拉满" |
| 邻家系 | 夸赞型 | "是让人看一眼就想带回家见妈妈的类型" |

## 付费详情页 (6 维度)

### 雷达图 (3 维)
- **少女感**: 胶原蛋白、苹果肌、饱满度
- **高级感**: 骨相、轮廓、留白感
- **氛围感**: 眼神、神态、气质

### 进度条 (3 项)
- **肤质状态**: 光泽、细腻度、均匀度
- **抗老指数**: 法令纹、眼周、紧致度
- **元气值**: 气色、黑眼圈、疲态

## 技术架构

### 前端技术栈
- Vite + React
- TailwindCSS v4
- shadcn/ui 组件库
- Framer Motion 动效

### API 集成
- Gemini 2.5 Flash (JSON Mode)
- 前端直调 (Demo 阶段)

### 页面结构
```
首屏: 分享卡片 (可保存/分享)
    ↓ 下滑
详情页: 付费解锁的详细分析
```

## JSON Schema 核心结构

```typescript
interface SkinScanResult {
  // 基础信息
  aura_type: "cool" | "sweet" | "queen" | "vibe" | "girl_next_door";
  aura_label: string;        // "清冷系 · 白月光"
  predicted_age: number;
  beauty_score: number;      // 0-100
  tagline: string;           // 一句话评语
  
  // 雷达图维度 (付费)
  radar: {
    youthfulness: number;    // 少女感 0-100
    elegance: number;        // 高级感 0-100  
    vibe: number;            // 氛围感 0-100
  };
  
  // 进度条维度 (付费)
  metrics: {
    skin_quality: number;    // 肤质状态 0-100
    anti_aging: number;      // 抗老指数 0-100
    vitality: number;        // 元气值 0-100
  };
  
  // 建议 (付费)
  recommendations: string[];
}
```
